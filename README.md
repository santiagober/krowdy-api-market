# Krowdy Api Market

Aplicacion tipo MercadoLibre o Ebay donde los usuarios pueden: crear productos, modificarlos, borrarlos y visualizar productos de otros usuarios.

## Tecnologias que usaremos:

- Express
- MongoDB
- Passport
- Hapi
